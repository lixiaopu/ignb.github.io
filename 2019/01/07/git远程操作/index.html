git远程操作

发表于 2019-01-07   |   更新于 2019-01-08

管理文件

添加文件到git仓库
$ git add file1.txt
$ git add file2.txt file3.txt
$ git commit -m “add 3 files.”

查看提交历史记录(过去)
$ git log

查看每一次命令历史(未来)
$ git reflog

HEAD指向的版本就是当前版本，上一个版本就是HEAD^，上上一个版本就是HEAD^^，往上100个版本写成HEAD~100
$ git reset –hard commit_id

查看当前状态
$ git status

在工作区乱改文件，但还没有add
$ git checkout – file

在工作区乱改文件，并且add到暂存区
$ git reset HEAD
$ git checkout – file

删除文件
$ git rm test.txt
$ git commit -m “remove test.txt”

恢复误删的文件
$ git checkout – test.txt

管理远程库

要关联一个远程库，使用命令
$ git remote add origin git@server-name:path/repo-name.git

关联后，第一次推送master分支的所有内容
$ git push -u origin master

此后，每次本地提交后，可以使用命令
$ git push origin master

从远程主机克隆一个版本库
$ git clone 版本库的网址

分支管理

创建并切换分支
$ git checkout -b dev
等同于
$ git branch dev
$ git checkout dev

查看当前分支
$ git branch

在dev上add，commit后回到master分支，合并到master，并删除dev
$ git checkout master
$ git merge dev
$ git branch -d dev
