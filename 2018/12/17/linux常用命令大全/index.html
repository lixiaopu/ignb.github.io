linux命令行删除文件或文件夹

发表于 2018-12-17   |   更新于 2018-12-20

语法
rm [options] 文件或文件夹名

option:
-i 删除前逐一询问确认。
-f 即使原档案属性设为唯读，亦直接删除，无需逐一确认。
-r 将目录及以下之档案亦逐一删除。

rm *
删除当前文件夹下所有文件，如果当前文件夹下有目录则用以下语句

rm -r *
删除当前文件夹下所有文件以及文件夹

rm 某文件.后缀
删除当前文件夹下某文件

rm -r 某文件夹
删除某文件夹

rm -rf 某文件或文件夹
直接删除，无需确认
